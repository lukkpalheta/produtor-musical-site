# 📧 COMO CONFIGURAR RECEBIMENTO DE EMAILS

## 🎯 Passo a Passo Simples

### **1. Acesse o Formspree**
- Vá para: https://formspree.io
- Clique em "Get Started"
- Crie uma conta gratuita

### **2. Crie um Novo Formulário**
- Clique em "New Form"
- Dê um nome: "Site Produtor Musical"
- Copie o ID do formulário (ex: xpzgkqwe)

### **3. Configure o Seu Site**
- Abra o arquivo index.html
- Procure por: SEU_FORM_ID_AQUI
- Substitua pelo ID do Formspree

### **4. Teste o Formulário**
- Abra o site no navegador
- Preencha o formulário de contato
- Envie uma mensagem de teste
- Verifique seu email

## 🔧 Configuração Atual

### **Formulário HTML:**
`html
<form action="https://formspree.io/f/SEU_FORM_ID_AQUI" method="POST">
    <input type="text" name="name" placeholder="Seu Nome" required>
    <input type="email" name="email" placeholder="Seu Email" required>
    <input type="text" name="subject" placeholder="Assunto" required>
    <textarea name="message" placeholder="Sua Mensagem" required></textarea>
    <button type="submit">Enviar Mensagem</button>
</form>
`

### **Campos Ocultos:**
- _next: Página de redirecionamento após envio
- _subject: Assunto do email recebido
- _replyto: Email para resposta

## 📱 Alternativas

### **EmailJS (JavaScript)**
1. Acesse: https://www.emailjs.com
2. Crie uma conta gratuita
3. Configure seu provedor de email
4. Use o código JavaScript fornecido

### **Netlify Forms**
1. Hospede no Netlify
2. Adicione "netlify" ao formulário
3. Emails chegam automaticamente
