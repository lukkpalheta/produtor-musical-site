2. Use Ctrl+F para buscar as informações atuais

4. Salve o arquivo

### **Método 2: Usando Busca e Substituição**
1. Abra o arquivo no Notepad
2. Pressione Ctrl+H
3. Digite a informação atual
4. Digite a nova informação
5. Clique em "Substituir Tudo"
